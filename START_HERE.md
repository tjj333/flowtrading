# IBKR data pull

Pull the instruments below from IBKR as 5-minute bars and save each as a local CSV. That's the whole job.

## Format
- 5-minute bars, **ADJUSTED** (splits + dividends)
- Columns: `datetime,open,high,low,close,volume` ; ISO-8601 timestamps with tz offset
- Window: **Jun 2025 → present** (or each name's listing start)
- US-listed: US cash session 09:30–16:00 ET. Filename `{TICKER}_5min_adjusted.csv`
- Tokyo ords (`.T`): TSE cash session 09:00–11:30 + 12:30–15:00 JST. Filename `{CODE}_TSEJ_5min_adjusted.csv`
  - **Must include the OPEN bars (09:00–09:30 JST) and CLOSE bars (14:30–15:00 JST) every day — full session, not first/last only.** (A prior pull had only midday/overnight bars and was unusable.)
